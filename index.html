Project Terbaru

Website Monitoring Kualitas Air IOT

Sebuah platform canggih untuk memantau dan menganalisis kualitas air secara real-time. Dengan fitur login untuk keamanan data, kemampuan eksport data ke Excel untuk kemudahan analisis lebih lanjut, update data otomatis atau manual, dan visualisasi data dalam bentuk chart yang informatif, situs ini memberikan kemudahan dalam mengelola dan memahami informasi kualitas air. Selain itu, pengguna dapat menyesuaikan tampilan dengan tema dark atau light sesuai preferensi untuk kenyamanan visual.

Situs web kami juga terintegrasi dengan alat IoT menggunakan mikrokontroler ESP32, mendukung berbagai sensor seperti sensor pH, TDS, suhu, dan kekeruhan untuk pemantauan kualitas air yang akurat dan real-time. Dengan website ini, dapat menjadi solusi lengkap untuk memastikan kualitas air tetap terjaga dan mudah dipantau.

Website Sistem Absensi Otomatis IOT

Website sistem absensi otomatis, solusi canggih untuk mengelola kehadiran dengan lebih efisien. Fitur login kami memastikan keamanan data dengan mengizinkan hanya pengguna terdaftar yang dapat mengakses sistem. Selain itu, pengguna dapat menambahkan data absensi secara manual, mengekspor data ke Excel untuk analisis lebih lanjut, serta mengedit dan menghapus data sesuai kebutuhan.

Sistem ini terintegrasi dengan alat IoT menggunakan mikrokontroler ESP32 dan sensor RFID, memungkinkan pencatatan kehadiran secara otomatis dan akurat. Dengan fitur-fitur ini, situs web kami memberikan kemudahan dalam pengelolaan absensi, memastikan data kehadiran tercatat dengan tepat dan mudah diakses kapan saja.

Website Monitoring Ruangan dan Deteksi Kebakaran IOT

Website monitoring ruangan dan deteksi kebakaran, solusi canggih untuk memastikan keamanan dan kenyamanan lingkungan Anda. Situs web kami menampilkan data real-time mengenai suhu dan kelembaban ruangan, serta status deteksi api dan gas berbahaya.

Dengan integrasi sensor DHT untuk memantau suhu dan kelembaban, sensor flame untuk mendeteksi api, dan sensor MQ2 untuk mendeteksi gas berbahaya, semuanya dikendalikan oleh mikrokontroler ESP32, kami memberikan pemantauan yang akurat dan cepat. Selain itu juga terdapat menu data record dimana setiap menit data yang dikeluarkan akan terekam.
